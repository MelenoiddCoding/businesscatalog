# API Spec

## Health
- `GET /health`
- Respuesta:
  - `status`: `ok`
  - `environment`: ambiente activo del backend (`local`, `preview`, `production`)
  - `app_name`: nombre visible de la app
  - `database_configured`: `true` si `DATABASE_URL` existe
