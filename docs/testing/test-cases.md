# Test Cases

## Casos criticos
- Registro exitoso con validacion de campos.
- Login exitoso y persistencia de sesion.
- Home con categorias y negocios destacados.
- Busqueda por texto con resultados y sin resultados.
- Filtro por categoria y zona combinados.
- "Cerca de mi" con permiso aceptado y denegado.
- Vista de mapa con marcadores y resumen de negocio.
- Ficha de negocio con datos completos y placeholders si faltan medios.
- Guardar y quitar favoritos autenticado.
- CTA de WhatsApp disponible solo con numero valido.
- Reseñas visibles y envio de reseña autenticado.
- Perfil con edicion basica de datos.
- Responsive mobile y desktop sin romper jerarquia visual.
